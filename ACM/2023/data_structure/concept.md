# 数据结构

@@

## why data structure?

- 数据结构是将数据高效集合和处理的工具
- 每种数据结构都各有优劣
    - 一种数据结构能够快速维护某些操作，而对其它的操作则比较低效
- 对不同的问题，选择不同的数据结构
- 抽象：功能；具体：实现

# 线性数据结构

@@

## 栈

栈是一种先进后出的数据结构，常见用法有括号匹配、表达式计算和单调栈等

C++ 中的栈：
- 声明：`stack<int> st`
- 加入元素 $a$：`st.push(a)`
- 栈顶元素：`st.top()`
- 弹出栈顶：`st.pop()`
